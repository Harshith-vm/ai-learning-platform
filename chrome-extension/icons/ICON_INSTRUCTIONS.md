# Extension Icon Instructions

## Required Icons

The Chrome Extension requires the following icon sizes:

- `icon16.png` - 16x16 pixels (toolbar)
- `icon32.png` - 32x32 pixels (toolbar retina)
- `icon48.png` - 48x48 pixels (extension management)
- `icon128.png` - 128x128 pixels (Chrome Web Store)

## Temporary Solution

For development and testing, you can:

1. **Use online icon generators**:
   - https://www.favicon-generator.org/
   - https://realfavicongenerator.net/
   - https://www.icoconverter.com/

2. **Create simple icons**:
   - Use any image editor (Photoshop, GIMP, Figma, Canva)
   - Create a square canvas
   - Add a simple design (e.g., "AI" text, brain icon, book icon)
   - Export in the required sizes

3. **Use emoji as icons** (quick method):
   - Take a screenshot of an emoji (🤖, 📚, 🧠, ✨)
   - Resize to required dimensions
   - Save as PNG

## Recommended Design

**Theme**: AI Learning / Education
**Colors**: Purple/Blue gradient (matching the popup UI)
**Symbol**: Brain, Book, or Robot icon

**Example concepts**:
- 🤖 Robot face (AI assistant)
- 📚 Book with sparkles (learning)
- 🧠 Brain icon (intelligence)
- ✨ Sparkle/star (enhancement)

## Quick Fix for Testing

If you need to test immediately without icons:

1. Create a simple colored square in any image editor
2. Save it as all four required sizes
3. Place them in this `icons/` folder
4. The extension will work fine for development

## Production Icons

For production release, create professional icons:

1. **Hire a designer** or use:
   - Fiverr
   - Upwork
   - 99designs

2. **Use icon design tools**:
   - Figma (free)
   - Adobe Illustrator
   - Sketch

3. **Follow Chrome Web Store guidelines**:
   - https://developer.chrome.com/docs/webstore/images/

## Color Scheme (from popup UI)

- Primary: `#667eea` (Purple-blue)
- Secondary: `#764ba2` (Purple)
- Accent: `#ffffff` (White)

Use these colors for consistency with the extension UI.
