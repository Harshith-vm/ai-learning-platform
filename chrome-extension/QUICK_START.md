# Quick Start Guide

## 🚀 Get Started in 5 Minutes

### Step 1: Create Icons (2 minutes)

1. Open `create-placeholder-icons.html` in your browser
2. Click "Download All Icons"
3. Save all 4 icons in the `icons/` folder

### Step 2: Install Extension (1 minute)

1. Open Chrome: `chrome://extensions/`
2. Enable "Developer mode" (top-right toggle)
3. Click "Load unpacked"
4. Select the `chrome-extension` folder
5. Done! ✅

### Step 3: Test It (2 minutes)

1. Go to any webpage (e.g., Wikipedia)
2. Select/highlight some text
3. Click the extension icon (toolbar)
4. Click "Summarize Selection"
5. See the alert: "Selected text captured!" ✅

## ✅ Success Checklist

- [ ] Extension icon appears in Chrome toolbar
- [ ] Popup opens when clicking icon
- [ ] Can select text on webpage
- [ ] "Summarize Selection" button works
- [ ] Alert appears with captured text
- [ ] Console shows logs

## 🐛 Troubleshooting

**Extension won't load?**
- Check all files are in `chrome-extension/` folder
- Verify icons exist in `icons/` folder
- Look for errors on `chrome://extensions/`

**No text captured?**
- Make sure text is highlighted (blue background)
- Refresh the webpage
- Check browser console (F12)

**Popup doesn't open?**
- Reload the extension
- Check for JavaScript errors
- Try disabling/re-enabling
